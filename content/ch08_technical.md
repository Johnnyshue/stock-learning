## Ch8 技術分析入門

### 什麼是技術分析？

技術分析 = 透過**股價走勢圖和成交量**來預測未來走勢。

基本面分析問「這家公司值多少錢？」，技術分析問「現在是買進還是賣出的好時機？」

> ⚠️ **注意**：技術分析的有效性有爭議，學術界普遍認為市場難以預測。建議作為輔助工具，不要作為唯一決策依據。

### K 線圖基礎

K 線（Candlestick）是最基本的技術分析工具：

```
    │        ← 上影線（盤中最高價）
   ┌┴┐
   │  │      ← 紅色實體（收盤 > 開盤 = 上漲）
   │  │         綠色實體（收盤 < 開盤 = 下跌）
   └┬┘
    │        ← 下影線（盤中最低價）
```

- **大紅 K**：強烈買盤
- **大綠 K**：強烈賣壓
- **十字線**：多空拉鋸，可能轉折

### 五個基本技術指標

**1. 移動平均線（MA）**
- 5MA = 5 日均價 → 短期趨勢
- 20MA = 20 日均價 → 月線，中期趨勢
- 60MA = 60 日均價 → 季線
- 240MA = 年線 → 長期趨勢
- **黃金交叉**（短均線上穿長均線）= 看多訊號
- **死亡交叉**（短均線下穿長均線）= 看空訊號

**2. 成交量（Volume）**
- 價漲量增 → 趨勢健康
- 價漲量縮 → 上漲無力，可能反轉
- 爆大量 → 注意可能是頭部或底部

**3. RSI（相對強弱指標）**
- 範圍 0-100
- RSI > 70 → 超買，可能回檔
- RSI < 30 → 超賣，可能反彈
- 不是絕對指標，強勢股可以在超買區待很久

**4. MACD**
- 由快線（DIF）和慢線（MACD）組成
- DIF 上穿 MACD → 買進訊號
- DIF 下穿 MACD → 賣出訊號
- 柱狀圖由負轉正 → 趨勢轉多

**5. 布林通道（Bollinger Bands）**
- 由中軌（20MA）和上下軌（±2 標準差）組成
- 股價觸及上軌 → 可能過熱
- 股價觸及下軌 → 可能超跌
- 通道收窄 → 即將出現大波動

### 技術分析的正確使用方式

1. **不要單看一個指標**——至少 2-3 個指標互相驗證
2. **配合基本面**——技術分析用來選擇進出場時機，基本面決定要不要買
3. **設定停損**——進場前就決定虧多少要跑（例如 -10%）
4. **承認錯誤**——技術分析不是 100% 準確，做錯就認賠
