## Ch0.6 指標背後的原理 + 怎麼自己比較兩支股票

> 給「只會看股價、不知道背後在幹嘛」的你。
> 這章不只告訴你指標「是什麼」，更講「**為什麼這樣看**」和「**會騙你的坑**」，
> 最後教你一張**評分卡**，自己判斷「哪支比較好」——不用問任何人。
> 📌 名詞定義在 [Ch0.2 九大指標](#00c)、[Ch0.3 技術面](#00d)；這章講更深的「原理」。

## 一、核心觀念：每個指標都在回答「一個問題」

| 指標 | 它其實在問 |
|---|---|
| EPS | 這公司「真的有賺錢嗎」？ |
| ROE / ROA | 它「賺錢的效率」高不高？是不是靠借錢撐？ |
| 本益比 / 殖利率 | 現在這價格「貴不貴、划不划算」？ |
| K線 / 量 / KD | 短期「多空誰佔上風、人氣旺不旺」？ |

> 股價只是「結果」。這些指標是「**為什麼會是這個股價**」的線索。只看股價 = 只看考試分數，不看他怎麼讀書。

---

## 二、基本面指標：背後原理 + 坑

### EPS（每股盈餘）——一切的地基
- **原理**：EPS = 公司淨賺的錢 ÷ 股數。它代表「**每一股幫你賺多少**」。
- **為什麼最重要**：長期來看，**股價會跟著 EPS 走**。公司持續多賺 → 股價長期向上；EPS 衰退 → 股價遲早跟著掉。
- 🕳️ **坑**：
  - 看「**連續好幾年**」的 EPS，不是只看一年（一年好可能是賣資產、業外收入的一次性）。
  - EPS 成長**比 EPS 高低更重要**：今年 5 元、去年 2 元（成長）比一直停在 8 元更有看頭。

### ROE（股東權益報酬率）——會賺錢，但小心「灌水」
- **原理**：ROE = 淨利 ÷ 股東權益（股東出的錢）。衡量「**用你的錢**賺錢的效率」。
- 🔑 **為什麼會被灌水（重要原理）**：
  公司去**借錢**來賺，淨利會變多，但「股東權益」（分母）沒變 → **ROE 被推高**。
  看起來很強，其實是**借來的槓桿**，景氣差、利率升時風險很大。
- 🕳️ **坑**：ROE 高，一定要再看 ROA。

### ROA（資產報酬率）——拆穿灌水的照妖鏡
- **原理**：ROA = 淨利 ÷ **總資產**（股東的錢 + 借來的錢）。
- 🔑 **為什麼能拆穿槓桿**：借來的錢會算進「總資產」（分母變大），所以**借錢救不了 ROA**。
- **判讀**：
  - ROE 高、ROA 也高 → 真本事 👍
  - **ROE 高、ROA 卻低 → 靠借錢撐的，要小心** ⚠️

### 本益比 PER / 殖利率 Yield——估值（貴不貴）
- 原理與口訣見 [Ch0.2](#00c)。一句話：本益比比「賺多少」、殖利率比「配多少」。
- 🕳️ **共同坑**：兩個都是「相對」指標，**只能跟「同產業」和「它自己過去」比**。半導體的本益比拿去比銀行毫無意義。

## 三、技術面指標：背後原理 + 坑

### K 線——一根棒子裡的「多空拉鋸」
- **原理**：每根 K 記錄一段時間的 **開盤、收盤、最高、最低**。
  - 收盤 > 開盤（台股紅 K）= 這段時間**買方贏**（把價格往上推）。
  - 上下影線 = 盤中「衝過但被打回」的痕跡，代表**有人在那個價位強力買/賣**。
- 🕳️ **坑**：單一根 K 線意義有限，要看「一串」的形狀和位置；K 線型態的「準度」學術上爭議大，別當鐵律。

### 成交量——人氣與「認同度」
- **原理**：量 = 這段時間「換手了幾張」。**有量＝有人真金白銀認同這個價格**。
- 🔑 **為什麼要「價量配合」**：
  - 價漲 + **量增** → 大家追，較可信（真漲）。
  - 價漲 + **量縮** → 沒人接，可能漲假的（小心追高套牢）。
- 🕳️ **坑**：爆大量不一定是好事——可能是「換手出貨」的頂部訊號。

### KD（隨機指標）——它在算「收盤站在近期高低的哪裡」
- **原理（很多人不知道）**：KD 其實在算「**今天收盤價，落在最近一段時間（常用 9 天）最高～最低區間的相對位置**」。
  - 收盤越靠近區間**高點** → K 值越高（接近 100）→ 近期偏強 → 「超買」。
  - 收盤越靠近區間**低點** → K 值越低（接近 0）→ 近期偏弱 → 「超賣」。
  - D 值是 K 的「平滑版」，比較不毛躁。
- 🕳️ **坑（一定要懂）**：
  - **鈍化**：強勢股的 K 可以**一直待在 80 以上**很久，這時「超買就賣」會太早下車。
  - KD 在「**橫盤震盪**」時較準，在「**單邊大趨勢**」時會一直發假訊號。

## 四、把「坑」整理成一張表（最容易踩的）

| 指標 | 最常見的坑 |
|---|---|
| EPS | 只看一年、忽略一次性收入、忽略「成長性」 |
| ROE | 高 ROE 沒看 ROA → 沒發現是借錢撐的 |
| ROA | 不同產業天生高低不同（銀行天生低），要同業比 |
| 本益比 | 低不一定便宜，可能是公司要衰退了 |
| 殖利率 | 高不一定好，可能是股價跌出來的「股息陷阱」 |
| K線 | 單根沒意義、型態不是鐵律 |
| 成交量 | 只看價不看量；爆量當成一定是好事 |
| KD | 鈍化（強勢股待在超買區）、趨勢盤一直假訊號 |

## 五、🎯 怎麼自己比較兩支股票：評分卡

拿 Goodinfo 的數字，照這張表幫每支打分，**分數高的「體質」較好**（但不等於「現在該買」）：

| 看的面向 | 加分（好） | 扣分（壞） |
|---|---|---|
| EPS | 連續多年正、且成長 | 忽正忽負、衰退 |
| ROE | > 15~20% | < 10% |
| ROA | > 8~10% | 很低（且 ROE 卻很高 = 槓桿） |
| 本益比 | 比同業 / 自己過去低 | 明顯偏高 |
| 殖利率 | 配息穩、是「配很多」 | 是「股價跌出來的」 |
| 產業/趨勢 | 長期需求向上 | 夕陽產業 |

### 假設範例（數字純虛構，只示範「怎麼想」）

> ⚠️ 下面 A、B 是**我編的假想公司**，不是真股票，只為教你判讀邏輯。

| 面向 | A 公司 | B 公司 |
|---|---|---|
| EPS | 連 5 年正、逐年成長 ✅ | 忽正忽負、去年轉虧 ❌ |
| ROE | 22% ✅ | 25%（更高！） |
| ROA | 12% ✅ | **只有 4%** ⚠️ |
| 本益比 | 15（同業中等） | 8（很低，好便宜？） |
| 殖利率 | 3% | **9%**（好高！） |

**新手直覺**：B 的 ROE 更高、本益比更低、殖利率超高 → B 比較好？

**會看的人怎麼想**：
- B 的 ROE 25% 但 **ROA 只有 4%** → 高 ROE 是**借錢撐**的（槓桿大、風險高）。
- B 本益比只有 8 → 不是便宜，是 **EPS 不穩、市場不看好**才被打到這麼低。
- B 殖利率 9% → 對照股價，很可能是**股價暴跌**造成的「股息陷阱」。
- A 雖然每個數字「沒那麼亮眼」，但 **穩、不靠槓桿、估值合理** → **體質明顯比 B 健康**。

> 🔑 **結論**：「哪支比較好」= 你照這張表**自己打分**得出的，不是別人報的。
> 而且「體質好」≠「現在買就會漲」——估值、時機、運氣都有份，所以才要 **分散（ETF）+ 定期定額 + 長期**。

### 信心等級備註
- 各指標的「原理與機制」（EPS 驅動長期股價、ROE 槓桿灌水、ROA 拆穿槓桿、KD 算相對位置、價量配合）→ **高信心**（財報/技術分析通用原理）。
- KD「常用 9 天」、ROE>15~20%、ROA>8~10% 等門檻 → **中信心**：常見參考值，非唯一標準，不同書/設定有差。
- A/B 公司數字 → **純虛構教學範例**，非任何真實股票。
- 技術指標（K線型態、KD）能否穩定獲利 → 學術證據偏弱，**當輔助、別當鐵律**（見 [Ch14](#14)）。
- 本章**不報任何真實個股的好壞或買賣建議**。
