## Ch7 基本面分析——主動選股

> ### 🐣 新手白話導讀
> **一句話**：這章教你怎麼「翻一家公司的帳本」，看它賺不賺錢、健不健康、貴不貴，來判斷它的股票值不值得買。
>
> **術語翻譯**
> - **基本面分析** = 研究一家公司的財務、競爭力、成長性，判斷股票值不值得投資（不是看股價漲跌猜方向）。
> - **三大財務報表** = 公司的健康檢查報告：損益表（賺不賺錢）、資產負債表（家底健不健康）、現金流量表（有沒有真的收到錢）。
> - **EPS（每股盈餘）** = 淨利除以總股數，也就是「每一股幫你賺多少錢」。
> - **本益比（P/E）** = 股價除以 EPS，數字越低代表越「便宜」，但要跟同業比才準。
> - **殖利率（Dividend Yield）** = 每股股利除以股價，代表「花這個價錢買，每年領回多少股利」；但別只看殖利率高就買。
> - **ROE（股東權益報酬率）** = 淨利除以股東權益，衡量公司用自己的本錢賺錢的效率，本章說 > 15% 算優秀。
> - **護城河（Moat）** = 公司能擋住對手的長期優勢（品牌、網路效應、規模、轉換成本、專利等），對手越難追上越好。
>
> **只記這幾句**：① 看公司先看三大報表——尤其「營業現金流必須為正」，確認它是真的有收到錢。② 估值指標（本益比、股價淨值比、殖利率、ROE、PEG）不能單看一個，也要跟同業比較。③ 巴菲特最重視「有護城河」的公司——能長期擋住競爭對手的優勢。投資都有風險，這些是分析方法，不是買賣保證。
>

### 什麼是基本面分析？

基本面分析 = 研究一家公司的**財務狀況、競爭優勢、成長潛力**，判斷它的股票是否值得投資。

就像你評估一個病人不能只看 vital signs，投資也不能只看股價。

### 三大財務報表速讀

**1. 損益表（Income Statement）**——公司賺不賺錢？

| 項目 | 意義 | 看什麼 |
|------|------|--------|
| 營收（Revenue） | 公司賣了多少錢 | 是否持續成長 |
| 毛利率 | (營收-成本)/營收 | 越高代表產品越有競爭力 |
| 營業利益率 | 扣掉營運費用後的利潤率 | 本業是否賺錢 |
| 淨利（Net Income） | 最終賺到的錢 | 是否穩定或成長 |
| EPS（每股盈餘） | 淨利 ÷ 總股數 | 每一股幫你賺多少錢 |

**2. 資產負債表（Balance Sheet）**——公司健不健康？

| 項目 | 意義 | 看什麼 |
|------|------|--------|
| 總資產 | 公司擁有的一切 | 規模大小 |
| 總負債 | 公司欠的錢 | 負債比率是否過高 |
| 股東權益 | 資產 - 負債 | 公司的「淨值」 |
| 流動比率 | 流動資產/流動負債 | > 1.5 較安全 |
| 負債比 | 總負債/總資產 | < 50% 較健康 |

**3. 現金流量表（Cash Flow Statement）**——公司有沒有真的收到錢？

| 項目 | 意義 | 看什麼 |
|------|------|--------|
| 營業現金流 | 本業產生的現金 | 必須為正！ |
| 投資現金流 | 花在設備、研發的錢 | 通常為負（代表在投資） |
| 融資現金流 | 借錢/還錢/發股利 | 看公司的融資策略 |
| 自由現金流 | 營業現金流 - 資本支出 | 公司真正「可自由運用」的錢 |

### 五個必看的估值指標

1. **本益比（P/E Ratio）**= 股價 / EPS
   - 越低越「便宜」，但要跟同業比
   - 台積電 P/E ~20，成長股可能 30-50

2. **股價淨值比（P/B Ratio）**= 股價 / 每股淨值
   - < 1 代表股價低於公司淨值（可能被低估）
   - 金融股常用此指標

3. **殖利率（Dividend Yield）**= 每股股利 / 股價
   - 台股高股息股通常 4-6%
   - 但不要只看殖利率高就買！

4. **ROE（股東權益報酬率）**= 淨利 / 股東權益
   - 巴菲特最愛的指標
   - > 15% 算優秀

5. **本益成長比（PEG）**= P/E / 盈餘成長率
   - < 1 可能被低估
   - 適合評估成長股

### 護城河理論（Moat）

巴菲特（Warren Buffett）最重要的投資概念——找有**護城河**的公司：

- **品牌護城河**：Apple、Coca-Cola → 消費者只認這個牌子
- **網路效應**：Meta、Visa → 越多人用越有價值
- **規模經濟**：台積電、Amazon → 規模大到對手追不上
- **轉換成本**：Microsoft Office、Oracle → 換掉太麻煩
- **法規/專利**：藥廠的專利保護、特許經營權
