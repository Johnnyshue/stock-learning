## Ch23 配偶 / 家庭 / 遺產規劃

> ### 🐣 新手白話導讀
> **一句話**：這章在講「等你賺到一筆錢之後，怎麼讓另一半看得懂、子女學會理財、走得突然時家人也知道錢在哪、怎麼合法少繳一點遺產/贈與稅，並保護資產不被醫療糾紛拖垮」。
>
> **術語翻譯**
> - **緊急預備金** = 放在隨時能領出來的錢（建議佔總資產 10-20%），失業或意外時先擋著，不用急著賣股票。
> - **股 / 債比例** = 你的錢有多少買「股票」（漲跌大、報酬高）、多少買「債券」（穩、報酬低）。本章建議用「100 − 平均年齡」當買股票的比例。
> - **複利** = 利息再生利息、像滾雪球；本章舉的例子是「30 年 7% 複利」，時間越久滾得越大。
> - **定期定額** = 每個月固定一筆錢、固定買進，不去猜高低點，靠紀律。
> - **複委託 / Firstrade** = 兩種「買外國股票」的管道；複委託是透過台灣券商代買，Firstrade 是美國的網路券商。
> - **遺產稅 / 贈與稅** = 過世後留給家人的錢要課的稅（遺產稅）；生前先送錢給家人也要課的稅（贈與稅）。本章列了 2026 年的免稅額與級距。
> - **信託** = 把錢交給合法機構（如銀行信託部）依你的指示代管、發給指定的人，常用於照顧未成年或特殊需求的子女。
> - **自益 / 他益信託** = 受益人是「你自己」叫自益；受益人是「別人（如子女）」叫他益。
>
> **只記這幾句**：① 寫一份「配偶看得懂的家庭金融指南」——帳戶在哪、怎麼領、找誰問，本章說它比保險更重要；② 趁中後期「逐年合法贈與」分散，遺囑（建議公證遺囑）和信託都「只能事先做、事後做不到」；③ 任何標榜「幫醫師逃稅／隱藏資產」「私人信託」的都當詐騙，具體規劃一定要找律師／會計師。
>
> ---（以下是完整內容）

> **20-30 歲建立紀律**（[Ch11](#11)）；**40-60 歲傳承思考**——這章是給那一刻的你寫的。
>
> 不一定現在動手，但**必須現在知道**——很多事**法律上**只能事先做，事後做不到。
>
> ⚠️ **本章為一般教育，非個人法律建議**。涉及具體規劃**必須**諮詢律師 / 會計師 / 信託專員。

### 🚨 為什麼醫師特別需要這塊

| 醫師特質 | 對應風險 |
|---|---|
| 高所得 → 高累積資產 | 遺產稅 / 贈與稅課徵範圍大 |
| 醫療糾紛責任 | 個人資產可能被執行 |
| 醫業時間長 | 配偶 / 子女不熟悉資產分布 |
| 自費 / 開業者 | 公司資產 vs 個人資產混用風險 |
| 高度專業 | 子女對資產配置完全不懂 |
| 突發風險 | 值班 / 通勤 / 過勞死案例存在 |

> **核心觀念**：你不在了，你的家人有沒有**指南**？

### 💼 一、配偶協同：投資不同步是大坑

#### 最常見 3 個夫妻投資衝突

| 衝突 | 表現 |
|---|---|
| 風險偏好不同 | 一個想全押科技股，另一個只敢定存 |
| 資訊不對稱 | 主導投資那方獨自決策，另一方完全不知 |
| 信任崩盤 | 一次空頭虧 30%，配偶質疑你的判斷 |

#### 預防 SOP

##### 1. 婚前 / 婚初**對齊期待**

至少對齊 3 件事：
- 緊急預備金佔總資產比例（建議 10-20%）
- 股 / 債比例（建議用「100 - 平均年齡」當基準）
- 每年回顧頻率（建議至少 1 次）

##### 2. **每年一次**家庭資產會議

**議程**：
1. 列出所有帳戶（券商 / 銀行 / 保險 / 房地產）
2. 計算總資產 / 負債 / 淨值
3. 對照原計畫看偏離
4. 討論未來一年大事（買房 / 生育 / 換工作）
5. **配偶獨立**確認**他/她**能在你不在時管錢

##### 3. 共同記錄「**家庭金融指南**」

寫一份**配偶看得懂**的文件，含：
- 所有帳戶清單 + 登入方式
- 每月固定支出 / 自動扣款項目
- 緊急聯絡人（會計師 / 律師 / 信託專員）
- 投資配置原則（**不是個股**，是**指數 + 配置**）
- 「如果我不在了，第一週該做什麼」流程

> 這份文件**比保險更重要**——保險賠錢給家人，**指南**讓家人知道**怎麼用**那些錢。

### 👶 二、子女財務教育：3 階段

#### 階段 1：學齡前（4-7 歲）

**核心觀念**：金錢有限、選擇有取捨。

具體做法：
- 給零用錢（每週固定，不超量）
- 教「儲蓄罐 vs 立刻買玩具」
- **不**給太多——**稀缺**是學習的前提

#### 階段 2：學齡（8-15 歲）

**核心觀念**：複利、儲蓄率、機會成本。

具體做法：
- 開「**子女帳戶**」（每月小額定存 / 兒童存款）
- 帶他看銀行存摺利息
- 教複利公式（Excel 玩**「30 年 7% 複利」**）

#### 階段 3：青少年（16-22 歲）

**核心觀念**：被動投資、紀律、避坑。

具體做法：
- 父母**示範**自己每月定期定額（讓他看你的扣款紀錄）
- 帶他開**第一個投資帳戶**（複委託 / Firstrade）
- **不**給他大筆錢——**親自賺第一份零用 / 工讀**才有意義
- 推薦讀物：Morgan Housel《致富心態》（[Ch18](#18) 銀級）

> 🚨 **避免**：給子女幾百萬「自由運用」=**通常**是毀人的開始（沒紀律經歷的高金額）。

### 📜 三、遺產規劃：法律基礎（台灣）

> **重要免責**：以下為**一般法律框架**，**具體規劃必須諮詢律師**。法規會修法。

#### 法定繼承順位（民法第 1138 條）

```
1. 直系血親卑親屬（子女、孫）
2. 父母
3. 兄弟姊妹
4. 祖父母
配偶：與上述各順位「共同繼承」
```

#### 遺產稅（遺產及贈與稅法）

**2026 年現行制度**（請查當下財政部公告）：
- 免稅額：**1,333 萬**（每人，2026 額度）
- 配偶扣除額：**553 萬**
- 直系血親卑親屬扣除額：**56 萬 / 人**
- 課稅級距：**10% / 15% / 20%**（累進）

> ⚠️ 數字**會隨物價指數調整**——以財政部公告為準。

#### 贈與稅

- **每人每年免稅 244 萬**（2026 額度，會調整）
- 超過部分課 10-20% 累進稅率
- **夫妻共贈**：等於每年 488 萬可贈與子女

> **節稅技巧**（合法）：醫師中後期**逐年**贈與子女股票 / 現金，**避免**一次性遺產稅高負擔。

#### 預立遺囑：**5 種法定形式**（民法第 1190-1198 條）

| 形式 | 特性 |
|---|---|
| 自書遺囑 | 全文親筆 + 簽名 + 日期；**最簡單**但易被質疑 |
| 公證遺囑 | 公證人 + 2 證人；**最不易質疑** |
| 密封遺囑 | 自書後密封 + 公證人 + 證人 |
| 代筆遺囑 | 3 證人之一代筆 + 全員簽名 |
| 口授遺囑 | 緊急情況；3 個月失效 |

**醫師建議**：**主治後**做**公證遺囑**——花 1 萬塊省下日後家屬糾紛百萬訴訟費。

### 🏛️ 四、信託：醫師中後期值得評估

#### 適合用信託的情境

- 子女**未成年**，擔心離世後資產被挪用
- 子女**特殊**（身心障礙、保護需求）
- 配偶不擅理財，需要**專業託管**
- 預防醫療糾紛資產追償（**事先**設立才有效）
- 想「**世代規劃**」（孫輩教育金）

#### 不適合的情境

- 資產規模小（管理費吃掉效益）
- 無特殊照顧需求
- 想「逃稅」——**現代信託對遺產稅幾乎無逃避空間**

#### 信託類型簡介

| 類型 | 特性 |
|---|---|
| 自益信託 | 委託人自己受益（生前資產管理） |
| 他益信託 | 受益人為他人（傳承、子女教育） |
| 公益信託 | 社會公益用途 |

> 🚨 **務必**找**金管會核准**的合法信託機構：銀行信託部 / 大型律師事務所信託部門。
> 不要碰**沒登記**的「**私人信託**」/「**家族財產管理顧問**」——詐騙集團近年新型詐騙。

### 🩺 五、醫師專屬：醫療糾紛資產保護

#### 風險現實

- 醫師執業期間**統計上**會碰到至少 1 次糾紛
- 多數是和解；少數判決下來金額**極大**（千萬級）
- **個人資產**可能被執行（民法 / 強制執行法）

#### 預防原則（**事先**做才有效）

##### 1. **執業責任險**（必備）

- 個人責任險（依專科風險而定，**外科 / 婦產 / 急診**最高）
- **足額**承保——不要省小錢

##### 2. **資產分離**

- 個人資產 vs 家庭資產，**分帳戶**管理
- 自有財產 vs 配偶名下資產（婚後財產約定可登記）
- 開業者：診所**設立公司**（有限公司），降低個人連帶責任

##### 3. **信託保護**（中後期）

- 把**主要**生活 / 教育金放入**他益信託**
- 信託資產原則上**不被**個人債務追償
- ⚠️ **必須事先**——糾紛發生**後**才設立 = 法院可撤銷

##### 4. **海外配置**（極端情境）

- 一定比例資產放在**海外帳戶**（美股 IB / 海外不動產）
- 不是逃稅，是**資產分散風險**
- 必須誠實申報（[Ch19](#19) 海外所得規則）

> ⚠️ 任何「**幫醫師逃稅 / 隱藏資產**」的服務 = 99% 詐騙，1% 違法。

### 📋 六、醫師人生階段傳承 SOP

#### PGY-R5：**配偶共識 + 緊急預備金**

- [ ] 跟配偶 / 重要伴侶**一次**對齊投資理念
- [ ] 寫第一版「**家庭金融指南**」
- [ ] 6 個月緊急預備金完成

#### 主治初期（30-40）：**保險 + 第一次贈與**

- [ ] [Ch24](#24) 保險完整建置（壽險 + 失能 + 實支實付）
- [ ] 開始**每年贈與**子女 244 萬（如已有子女）
- [ ] 開立**子女投資帳戶**

#### 主治中期（40-50）：**遺囑 + 信託評估**

- [ ] 公證遺囑
- [ ] 評估是否設立家族信託
- [ ] 主要資產分離（個人 vs 家庭 vs 公司）

#### 主治後期（50+）：**完整傳承**

- [ ] 完整稅務 + 法律檢討（每 3 年一次）
- [ ] 子女金融教育**已完成**
- [ ] 配偶**獨立**能管理所有資產
- [ ] 緊急應變方案**書面化**

### 💡 一句話總結

> **錢留下來只是第一層；讓家人「**知道怎麼處理**」才是第二層。**
>
> 多數醫師卡在第一層——資產很多但配偶不知道帳戶在哪、子女不知道怎麼提領、糾紛時沒律師可問。
>
> **「家庭金融指南」是醫師最該寫但最少人寫的文件**。

---

### ⚠️ 必須諮詢專業的時機

- 主治後**首次**遺產規劃 → 律師 + 會計師
- 設立信託 → 銀行信託部 / 律師事務所
- 跨境資產配置 → 國際稅務專家 + 移民法律師
- 醫療糾紛發生 → **立刻**律師
- 公司設立 / 解散 → 會計師
