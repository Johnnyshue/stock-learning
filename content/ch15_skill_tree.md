## Ch15 股票達人能力樹（5 大支柱 × 青銅→王者）

> [Ch11 路線圖](#11) 是**時間軸**（when：第 1 年做什麼、第 5 年做什麼）。
>
> 這章是**能力樹**（what skills：你需要的所有能力分成哪幾類，每類目前在哪個段位）。
>
> 兩章配合用：先看能力樹知道**自己缺什麼**，再回 [Ch11](#11) 找**什麼時候補**。

---

### 🎯 為什麼用「能力樹」？

> ⚠️ **「股票達人」不是一個技能，是 5 個獨立技能的組合。**
>
> 多數散戶錯在**只練一個**（例如只練技術分析），其他 4 個 0 分 → 結果還是輸。

---

### 🌳 5 大能力支柱總覽

```
                  股票達人
                     │
    ┌────────┬───────┼───────┬────────┐
    │        │       │       │        │
  🧠 知識  🛠️ 工具  🧘 心理  📋 行為  🛡️ 防禦
   (學)    (用)    (忍)    (做)    (防)
```

| 支柱 | 核心問題 | 失敗代價 |
|---|---|---|
| 🧠 **知識** | 你**懂不懂**金融市場？ | 不懂 → 被話術騙 |
| 🛠️ **工具** | 你會不會**用工具**驗證？ | 不會 → 仰賴別人嘴巴 |
| 🧘 **心理** | 你能不能**忍住**情緒衝動？ | 忍不住 → 高買低賣 |
| 📋 **行為** | 你**實際做了什麼**？ | 沒做 → 永遠在學 |
| 🛡️ **防禦** | 你能否**避開**陷阱？ | 不能 → 一次破產 |

> **任何一個支柱卡在青銅 = 整個系統薄弱。**

---

### 🧠 支柱 1：知識（學）

**目標**：理解市場運作機制，能識別錯誤論述。

| 段位 | 能力描述 | 對應章節 |
|---|---|---|
| 🥉 **青銅** | 知道 ETF 是什麼、ETF 跟個股的差別 | [Ch1](#01)、[Ch5](#05) |
| 🥈 **白銀** | 看得懂財報三大表（損益、資產負債、現金流）基本欄位 | [Ch7](#07) |
| 🥇 **黃金** | 理解 EMH、被動 vs 主動的學術爭論、SPIVA 數據意義 | [Ch5.2](#05c)、[Ch14](#14) |
| 💎 **白金** | 能用三層框架（基本 / 資金 / 情緒）解讀市場新聞 | [Ch13](#13) |
| 👑 **王者** | 能對任何 narrative 做反駁，引用具體數據與學術文獻 | 自學深化 |

**自我檢測**：
- [ ] 我說得出「為什麼 95% 主動基金跑輸指數」的學術原因
- [ ] 看到「保證獲利」就立刻知道是詐騙
- [ ] 能解釋「為什麼短期股價接近隨機」

### 🛠️ 支柱 2：工具（用）

**目標**：能獨立查證資料，不依賴他人轉述。

| 段位 | 能力描述 | 工具 |
|---|---|---|
| 🥉 **青銅** | 會用券商 App 下單、查持股 | 富邦 / 國泰 / 永豐證券 App |
| 🥈 **白銀** | 會用 MOPS 查財報、Goodinfo 查歷年數據 | [MOPS](https://mops.twse.com.tw/) / Goodinfo |
| 🥇 **黃金** | 會看法說會逐字稿、會查三大法人買賣超、會看券商研究報告 | [TWSE 法說會](https://mops.twse.com.tw/mops/web/t146sb05) |
| 💎 **白金** | 會用 Excel / Google Sheets 算 IRR、Sharpe、回測 | Excel `=IRR()`、Python `yfinance` |
| 👑 **王者** | 能自己跑 Python / R 做回測、因子分析、Monte Carlo 模擬 | Python `pandas`、`backtrader` |

**自我檢測**：
- [ ] 我會打開 MOPS 查任一上市公司的最新季報
- [ ] 我會用試算表算「定期定額 20 年」的最終金額
- [ ] 看到媒體引用數字時，我會去找原始來源

### 🧘 支柱 3：心理（忍）

**目標**：在市場波動中保持紀律，不被情緒帶走。

> 這是**最難練的一根**，但也是**威力最大的一根**。
> Buffett 說：「投資不需要高 IQ，需要的是**情緒紀律**。」

| 段位 | 能力描述 | 通過考驗 |
|---|---|---|
| 🥉 **青銅** | 帳面虧 -10% 不會立刻賣掉 | 一次跌幅 |
| 🥈 **白銀** | 帳面虧 -20% 仍照原計畫定期定額 | 一次大跌 |
| 🥇 **黃金** | 經歷一次完整空頭循環（-30%+）沒中斷紀律 | 一次熊市（如 2022 / 2008 / 2000） |
| 💎 **白金** | 多頭時忍住不加碼超出計畫，空頭時忍住不停扣 | 一次完整循環 |
| 👑 **王者** | 完全不被新聞 / 朋友 / 老師影響，自己有清晰投資哲學 | 多次循環 |

**自我檢測**：
- [ ] 我可以承受 -30% 帳面虧損不失眠
- [ ] 看到「老師」群組的喊單訊息我會直接刪掉
- [ ] 朋友炫耀短線賺翻時我不會羨慕

> ⚠️ **心理段位無法靠「讀書」升級**，只能靠**實際經歷市場波動**。所以從**金額小到不痛**開始最關鍵。

### 📋 支柱 4：行為（做）

**目標**：把知識轉成實際持續的動作。

| 段位 | 能力描述 | 具體動作 |
|---|---|---|
| 🥉 **青銅** | 已開戶 + 有第一筆投資（任何金額） | 開戶、買第一張 0050 |
| 🥈 **白銀** | 已設定**自動扣款**定期定額 ≥ 6 個月 | 券商自動扣款 |
| 🥇 **黃金** | 有書面**資產配置計畫**（股 / 債 / 現金比例） | 寫一份配置表 |
| 💎 **白金** | 每年**再平衡 1 次**（rebalance） | 一年 1 次調整比例 |
| 👑 **王者** | 完整稅務規劃 + 跨國配置 + 退休帳戶最佳化 | 美股配置 + 健保補充費規劃 |

**自我檢測**：
- [ ] 我有**自動定期定額**設定，不靠每月手動執行
- [ ] 我寫得出我的**資產配置目標**（例：股 80% / 債 20%）
- [ ] 我**過去 12 個月**有實際投資紀錄

> 重點：**段位 = 你實際做的事**，不是你**知道**或**打算**做什麼。

### 🛡️ 支柱 5：防禦（防）

**目標**：辨識並避開所有可能讓你**一次破產**的陷阱。

| 段位 | 能力描述 | 對應章節 |
|---|---|---|
| 🥉 **青銅** | 不買投資型保單、不買連動債 | [Ch12](#12) |
| 🥈 **白銀** | 不加任何「老師帶單」LINE 群、FB 投資社團 | [Ch12](#12) |
| 🥇 **黃金** | 認得 [Ch12](#12) 全部 6 種詐騙樣本，能教家人辨識 | [Ch12](#12) |
| 💎 **白金** | 看到任何「保證」「內線」「私訊」立刻轉身 | [Ch14](#14) |
| 👑 **王者** | 能識別所有變形版詐騙（含學長介紹、新形式幣圈、未上市股權） | 持續更新 |

**自我檢測**：
- [ ] 我**完全不**碰投資型保單 / 連動債 / 加密貨幣短線
- [ ] 我**完全不**進「老師帶單」LINE 群
- [ ] 我會用 [SITCA](https://www.sitca.org.tw/) 查投顧合法性
- [ ] 看到「保證高報酬」我會立刻想到「100% 詐騙」

> ⚠️ 這是**唯一**「卡在青銅就足以毀掉你財富」的支柱。**永遠不要忽略防禦**。

### 💉 醫師專屬加成 / 扣分項

| 項目 | 加成 / 扣分 | 影響 |
|---|---|---|
| **高且穩定收入** | +++ 加成 | 儲蓄率武器（[Ch11](#11) 詳述） |
| **時間結構不允許盯盤** | +++ 加成 | 被迫做被動投資 = 勝率最高策略 |
| **生技醫療專業** | + 加成 | 看得懂醫材 / 製藥公司產品週期 |
| **高自信** | --- 扣分 | 「醫學院都讀得起來」→ 跑去做主動交易 |
| **社交圈封閉** | --- 扣分 | 學長介紹的「好機會」最容易卸下心防 |
| **忙碌沒時間查證** | -- 扣分 | 容易聽信理專推坑 |
| **內線交易誘惑** | --- 扣分 | 違法 + 一次足以毀掉整個職業生涯 |

> **醫師的優勢全在「結構」，劣勢全在「心態」。**
> 這就是為什麼 [Ch10](#10) 和 [Ch11](#11) 反覆強調「**避開陷阱比追求績效更重要**」。

### 🎯 升級路徑：從青銅到王者

#### 🥉 青銅 → 🥈 白銀（0-1 年）

**重點**：**先做後學**——開戶 + 第一筆投資 + 設定自動扣款。

- 讀 [Ch0-Ch6](#00)（基本心法 + 開戶）
- 開複委託或美股帳戶（[Ch3](#03)、[Ch4](#04)）
- 第一筆 1-3 萬買 0050 或 VT
- 設自動定期定額
- 退掉所有 LINE 投資群

#### 🥈 白銀 → 🥇 黃金（1-3 年）

**重點**：**經歷波動**——通過第一次帳面 -10% 到 -20% 考驗。

- 讀完 [Ch7-Ch9](#07)（基本面、技術面、配置）
- 讀完 [Ch5.1-Ch5.3](#05b)（基金完整知識）
- 讀完《漫步華爾街》
- 寫第一份完整資產配置表
- 經歷一次帳面虧損 ≥ -10% 沒賣

#### 🥇 黃金 → 💎 白金（3-7 年）

**重點**：**完整循環**——通過一次空頭市場（-30%+）。

- 讀完 [Ch10-Ch12](#10)（醫師 / 路線圖 / 詐騙）
- 讀完 [Ch13-Ch14](#13)（市場 narrative + EMH）
- 經歷一次完整空頭循環沒中斷紀律
- 開始每年再平衡 1 次
- 完整稅務規劃（健保補充費 / 海外所得 / 美股股息扣繳）

#### 💎 白金 → 👑 王者（7 年+）

**重點**：**多重循環 + 教導他人**。

- 經歷 ≥ 2 次完整多空循環
- 能用三層框架解讀任何市場新聞
- 能教家人 / 同事 / 學生分辨假達人
- 自己跑回測、做因子分析（Python / R）
- **能對自己說：「我已不需要任何老師。」**
