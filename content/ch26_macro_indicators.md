## Ch26 全球宏觀指標解讀

> [Ch13](#13) 教你解讀**個別市場新聞**；這章教你**讀懂**全球經濟大格局——GDP / CPI / 利率 / 失業 / PMI。
>
> ⚠️ 一樣強調：**這是「解讀框架」，不是「預測工具」**（不要違反 [Ch14](#14) 的 EMH 教訓）。
>
> ⚠️ **所有數據必須查官方即時公告**，本章只給**框架 + 來源**，不寫具體數字。

---

### 🎯 為什麼醫師需要懂宏觀

| 不懂宏觀 | 懂宏觀 |
|---|---|
| 看新聞「央行升息 1 碼」沒感覺 | 知道對股 / 債 / 房地產的可能影響 |
| 「**通膨高**」聽過但不知怎麼解讀 | 知道**意外通膨**才影響市場 |
| 完全不看數據 → 全靠 narrative | 自己看官方數據 → 不被新聞操縱 |
| 「老師說經濟要崩了」就賣股 | 看完整數據判斷是否雜訊 |

> **目標不是預測**——是**抗雜訊**。

### 📊 一、GDP（國內生產毛額）

#### 是什麼

國家**一年內**生產的所有商品 + 服務的總價值。

#### 怎麼讀

| 看什麼 | 意義 |
|---|---|
| **GDP 年增率** | 經濟成長速度（一般 ≥ 3% 強，< 1% 弱） |
| **實質 vs 名目** | **實質** GDP 才是真正成長（扣掉通膨） |
| **GDP 構成**（消費 / 投資 / 政府 / 淨出口）| 看是「靠什麼撐」 |
| **環比 vs 年比** | 環比（QoQ）反映**最新動向**，年比（YoY）反映**整年趨勢** |

#### 對市場的影響

- GDP **超預期**強 → 一般利好股市 / 利空債（暗示央行可能升息）
- GDP **超預期**弱 → 利空股市 / 利好債（暗示央行可能降息）
- **預期之內** → 市場通常無反應

#### 官方來源

| 國家 / 區域 | 來源 |
|---|---|
| 台灣 | [主計總處（DGBAS）](https://www.dgbas.gov.tw/) |
| 美國 | [Bureau of Economic Analysis (BEA)](https://www.bea.gov/) |
| 全球 | [World Bank](https://data.worldbank.org/) / [OECD](https://data.oecd.org/) |
| 歐盟 | [Eurostat](https://ec.europa.eu/eurostat) |

### 💰 二、CPI（消費者物價指數，通膨）

#### 是什麼

代表**一般消費者**買的一籃子商品 + 服務的價格變動。

#### 為什麼**最重要**

通膨**直接**決定：
- **央行貨幣政策**（升息 / 降息）
- 你的**實質購買力**
- 你的**投資實質報酬**（名目 7% - 通膨 3% = 實質 4%）

#### 怎麼讀

| 看什麼 | 意義 |
|---|---|
| **整體 CPI** | 含食物 + 能源（波動大） |
| **核心 CPI** | 排除食物 + 能源（央行**最看**這個） |
| **YoY**（年增率）| 標準通膨指標 |
| **MoM**（月增率）| 短期動向 |

#### 央行通膨目標

- 美國 Fed：**2% 通膨目標**
- 台灣央行：**約 2%**（無嚴格通膨目標制）
- 歐洲央行：**2%**

> **超過目標 → 升息壓力**；**低於目標 → 降息空間**。

#### 對市場的影響

| CPI 表現 | 一般反應 |
|---|---|
| **意外**高 | 利空股 / 利空債 / 利空房地產（升息預期） |
| **意外**低 | 利好股 / 利好債 / 利好房地產（降息預期） |
| **預期之內** | 通常無顯著反應 |

> 關鍵是「**意外**」，不是絕對水準（[Ch14](#14) EMH——已預期的會 price in）。

#### 官方來源

| 國家 | 來源 |
|---|---|
| 台灣 | [主計總處 CPI](https://www.dgbas.gov.tw/) |
| 美國 | [BLS（Bureau of Labor Statistics）](https://www.bls.gov/cpi/) |

### 🏦 三、利率（央行政策利率）

#### 是什麼

央行**放給銀行**的基準利率——影響**所有**市場利率。

#### 各國名稱

| 國家 | 政策利率名稱 |
|---|---|
| 美國 | Federal Funds Rate（聯邦基金利率）|
| 台灣 | 重貼現率 |
| 歐洲 | 主要再融資利率 |
| 日本 | 隔夜拆款利率 |

#### 怎麼讀

| 看什麼 | 意義 |
|---|---|
| **當下利率** | 絕對水準 |
| **預期路徑**（CME FedWatch）| 市場猜下次決議 |
| **點陣圖**（Fed dot plot）| Fed 官員自己的預測 |
| **聲明 + 記者會** | 措辭**比數字重要** |

#### 對市場的影響

| 升息 | 降息 |
|---|---|
| 風險資產（成長股、加密貨幣、房地產）受壓 | 風險資產受惠 |
| 公債價格下跌（**反向**） | 公債價格上漲 |
| 美元走強 | 美元走弱 |
| 高息儲蓄產品變吸引 | 現金 / 定存報酬下降 |

#### 官方來源

- [FOMC 會議行事曆](https://www.federalreserve.gov/monetarypolicy/fomccalendars.htm)
- [CME FedWatch Tool](https://www.cmegroup.com/markets/interest-rates/cme-fedwatch-tool.html)
- [台灣央行](https://www.cbc.gov.tw/)

### 👷 四、失業率 + 就業數據

#### 是什麼

勞動力市場的健康指標。

#### 為什麼重要

- 失業率**過低** → 工資上漲 → 通膨壓力 → 升息可能
- 失業率**過高** → 經濟衰退訊號 → 降息可能

#### 美國最受關注的就業報告

**Non-Farm Payroll（非農就業）**——每月第一個週五公布。

| 看什麼 | 意義 |
|---|---|
| 新增非農就業人數 | > 25 萬強，< 10 萬弱 |
| 失業率 | < 4% 緊俏，> 5% 鬆弛 |
| 平均時薪年增率 | 通膨壓力指標 |

#### 對市場的影響

非農**意外**強 → 升息壓力 → 一般利空股市
非農**意外**弱 → 降息預期 → 一般利好股市

> **2022-2024**：「**好消息變壞消息**」（強就業反而股市跌，因為強化升息預期）。

#### 官方來源

- 美國：[BLS Employment Situation](https://www.bls.gov/ces/)
- 台灣：[主計總處 失業率](https://www.dgbas.gov.tw/)

### 🏭 五、PMI（採購經理人指數）

#### 是什麼

調查**製造業 + 服務業**採購經理人對景氣的判斷。

#### 怎麼讀

```
PMI > 50：景氣**擴張**
PMI = 50：景氣**持平**
PMI < 50：景氣**收縮**
```

#### 為什麼重要

- **領先**指標——**比** GDP **早 1-2 個月**反映景氣
- 各國 PMI 同步看 → 全球製造業景氣

#### 主要 PMI

| 來源 | 涵蓋 |
|---|---|
| **ISM Manufacturing PMI** | 美國製造業 |
| **ISM Services PMI** | 美國服務業 |
| **S&P Global PMI** | 全球版本 |
| **中國官方 PMI / 財新 PMI** | 中國（兩個略有差異） |
| **台灣 IHS PMI** | 台灣 |

#### 來源

- [ISM 官網](https://www.ismworld.org/)
- [S&P Global PMI](https://www.pmi.spglobal.com/)

### 💱 六、匯率

#### 對醫師影響

醫師持有**美股 ETF**：
- **新台幣升值** → 換回台幣**虧匯差**
- **新台幣貶值** → 換回台幣**賺匯差**

> **長期**（10+ 年）匯率波動往往**抵消**——不必太執著短期匯差。

#### 看什麼

| 指標 | 意義 |
|---|---|
| 美元指數（DXY）| 美元相對主要貨幣強弱 |
| USD/TWD | 美元兌台幣 |
| EUR/USD | 歐元兌美元 |
| 央行外匯干預 | 台灣央行常出手「**平準**」匯率 |

### 🧠 七、怎麼把宏觀指標**串起來**讀

#### 框架：通膨 + 成長兩維分析

```
              成長強
                ↑
                │
通膨低 ────────┼──────── 通膨高
                │
                ↓
              成長弱
```

| 區域 | 對應市場特徵 |
|---|---|
| **強成長 + 低通膨**（理想） | **股票黃金期**（如 2010s 美股） |
| **強成長 + 高通膨** | 央行升息 → 後續壓力 |
| **弱成長 + 低通膨** | 央行降息 → 風險資產回穩 |
| **弱成長 + 高通膨**（停滯性通膨）| 最糟，**1970s 美股** |

#### 對你的意義

**不必預測**現在在哪一象限——而是**在每個象限你的配置如何表現**：
- 你已經做的**全球分散 ETF + 股債配置**已**抗通膨**（股票 + 不動產）+ **抗成長**（債券）
- **配置紀律 = 不需要看宏觀也能存活**

### 🚨 八、看宏觀指標常見的 5 個錯誤

#### 錯誤 1：看單一指標就下結論

「失業率高 → 經濟差 → 我要砍股」
**真相**：失業率是**落後**指標；股市可能已反映或反向走。

#### 錯誤 2：被 narrative 綁架

「**新聞說經濟要崩**」 → 砍股
**真相**：新聞**最常**錯時點。

#### 錯誤 3：忽略「**意外**」概念

「失業率還是 4%」 → 沒事
**真相**：如果**預期**是 3.8%，4% 就是利空（實際**比預期差**）。

#### 錯誤 4：看絕對數字不看趨勢

「CPI 3% 是高還是低？」
**真相**：要看**趨勢**——3% 從 5% 降下來是好；從 1% 升上來是壞。

#### 錯誤 5：把宏觀當**選股工具**

「GDP 強 → 我買電子股」
**真相**：宏觀只能**側面**輔助配置決策，**不能**告訴你買哪檔。

### 💉 醫師專屬：宏觀 vs 紀律

| 你想做 | 該做的 |
|---|---|
| 看 CPI 高就賣股、低就買股 | ❌ 擇時失敗率高 |
| 看 Fed 升息就減倉 | ❌ 多半已 price in |
| 看 GDP 衰退就停扣 | ❌ 退休的人最需要紀律 |
| **理解**為什麼新聞這樣寫 | ✅ 抗雜訊 |
| **驗證**「老師」說的有沒有道理 | ✅ 提升判斷力 |
| **解釋**給家人聽，**安撫**配偶 | ✅ 通過空頭的關鍵 |

> 宏觀知識的**最大用途**：**不被別人嚇到改變紀律**。
