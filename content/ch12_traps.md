## Ch12 醫師專屬投資陷阱與詐騙樣本庫

> 醫師是詐騙集團的「**理想目標**」：高薪、高自信、忙碌、社交圈封閉、教育程度高（自認不會被騙）。
> 這章把所有針對醫師的陷阱蒐集起來，**看過一次永遠免疫**。

---

### 🔴 為什麼醫師特別容易被騙？

| 因素 | 詐騙集團怎麼利用 |
|---|---|
| 高薪可動用資金大 | 一次性入金 100 萬 / 200 萬不痛不癢 |
| 自認聰明 | 「我醫學院都讀得起來，這個應該也難不倒我」→ 心態漏洞 |
| 忙碌沒時間查證 | 「老師說有單就跟」→ 省下研究時間的代價 = 本金 |
| 社交封閉 | 醫師圈的 LINE 群、學長學姊推薦最容易卸下心防 |
| 高學歷自尊 | 被騙了也不好意思承認 → 詐騙集團敢挑醫師下手 |

### 🚨 詐騙樣本庫（持續更新）

#### Sample 1：FB 公開社團「股票投資交流群」型

**抓到日期**：2026-04-29
**社團名**：股票投資交流群（1.4 萬人）/ 股票研究俱樂部（1.6 萬人）

**特徵**：
- 社團名取「投資交流」「股票研究」「飆股研究」這類正常名稱
- 100% 貼文都是業配，**沒有任何真實散戶討論**
- 共同 LINE ID（如 `@465ugffe`、`1893929`）多帳號操作
- 後見之明吹噓：「我買台積電在 980 元漲到 2330」「我買聯發科在 980 漲到 2675」

**話術範例（原文）**：
```
散戶們不要虧錢，聽好了
到時候別又說怎麼沒人提醒
盤面亂追太多，重點群組直接點出
這幾檔很可能成為盤面焦點，一定要盯緊：
[顯示更多]
添加賴 ID :@465ugffe 進群組免費領取亮燈股！
```

**辨識要點**：
- 🚨 任何文章結尾要你「加 LINE / 私訊 / 入群」=  **100% 詐騙**
- 🚨 「我買在 X 漲到 Y」這類事後吹噓 = 完全無法驗證的話術
- 🚨 「亮燈股」「飆股」「報牌」「點明牌」全是詐騙黑話

#### Sample 2：偽專業派——「拿鐵選好股」型

**特徵**：
- 看似專業：每天寫 500-1000 字盤後分析（油價、地緣政治、外資動向）
- 文末固定句：「需要參考帶上車的同學 私訊 +88」「私訊 +666 進群」

**話術結構**：
1. 開頭：宏觀分析（伊朗、美聯儲、台股動向）→ **建立專業形象**
2. 中段：個股觀察（旺宏、聯發科、ETF）→ **放餌**
3. 結尾：「需要 XXX 私訊我」→ **收網**

**辨識要點**：
- 🚨 真正專業的人**不需要透過私訊推薦個股**——他們發 PR 報告、上節目、寫書
- 🚨 「+88」「+666」這類迷信數字 = 詐騙集團專用代號
- 🚨 一旦私訊就會被拉進付費群組，「老師」喊單，群員對作

#### Sample 3：醫師圈 LINE 群——「學長介紹的好機會」型

**典型場景**：
- 學長/學姊在科室 LINE 群分享：「我朋友 OOO 的群組績效很好」
- 進群後是「老師帶單」、「股票顧問」、「私募基金」
- 入金後被「教導」買特定個股，價格被群內對作拉抬，最後砸盤割韭菜

**為什麼這招最毒**：
- 醫師圈關係緊密 → **學長推薦自帶信任 buff**
- 你不好意思質疑學長 → 心理防線崩潰
- 進群後群內「績效見證」「老師回答 Q&A」全是內鬼演出

**辨識要點**：
- 🚨 任何「私募」「VIP 群」「老師帶單」**不論誰介紹**都拒絕
- 🚨 真正的合法投顧都有金管會核准執照（[查詢連結](https://www.sitca.org.tw/)）
- 🚨 「績效見證」截圖 100% 可造假

#### Sample 4：投資型保單——「兼具保障與投資」型

**典型場景**：
- 理專/業務員：「醫師月薪這麼高，可以買投資型保單」
- ILP（投資型保單）、變額萬能壽險、變額年金

**為什麼是陷阱**：
- 前 5 年保費 60-80% 被吃光（管銷費用、業務員佣金）
- 連動債或基金常配置爛標的（手續費高、IRR 低）
- IRR 普遍 < 通膨 → **負實質報酬**
- 「保障 + 投資」的兩個目標都做得比分開做差

**正確做法**：
- 純保障 → 買**定期壽險 + 失能險 + 醫療險** ← Ch10 已述
- 投資 → 自己買 0050 / VT，**完全不要混在一起**

#### Sample 5：加密貨幣 / 海外平台「保證高報酬」型

**特徵**：
- 「USDT 套利」「DeFi 挖礦」「海外交易所做合約」
- 標榜「日報酬 1%」「月化 30%」「保證不會虧」

**結局 100% 是**：
- 入金可入、出金不行
- 平台「升級維護」突然消失
- 客服失聯、域名解析失敗、跑路

**辨識要點**：
- 🚨 「保證」+「高報酬」 = **必詐騙**（金融鐵則）
- 🚨 要求換成 USDT/BTC 入金 = **完全追不到金流**
- 🚨 面交現金更是直接送錢

> 引用自 [Mobile01 真實詐騙案例](https://www.mobile01.com/topicdetail.php?f=291&t=7215188)：「只要符合下列其中一項，100% 就是詐騙：1. 要你換成加密貨幣才能投資 2. 入金要求面交現金。上面二種手法都是完全追不到金流。」

#### Sample 6：飆股 / 明牌簡訊轟炸

**特徵**：
- 莫名收到簡訊「我是 XXX 投顧，本週飆股 OOOO（個股代號）」
- 預測結果 50% 對、50% 錯（因為他們同時發兩種版本給不同人）
- 對的人會收到後續「我之前說對了，這次要不要進群繼續跟」

**辨識要點**：
- 🚨 **未訂閱卻收到投資簡訊** = 詐騙
- 🚨 預測 1-2 次對 ≠ 真實能力（這叫 **survivorship bias**）

### 💉 醫師專屬陷阱（不是詐騙但很雷）

#### 雷 1：被同事邀請當「醫療新創股東」

**情境**：學長創業做醫療 AI / 健康管理 App，邀你投資 100 萬當原始股東。

**問題**：
- **未上市股票流動性 = 0**——你想賣根本沒人接
- 估值靠老闆嘴喊，財報沒揭露義務
- 99% 新創 5 年內倒閉
- 即使 IPO 成功，掛牌後 lock-up period 限制更難退出

**判斷原則**：
- 投資金額 < 你能完全 write off 的閒錢
- 不要把「相信朋友」跟「投資判斷」混為一談
- 你不是早期 VC，沒有風險分散

#### 雷 2：醫師專屬「房地產投資講座」

**情境**：「醫師包租公」「醫師置產俱樂部」邀你聽課，主打高槓桿購屋出租。

**問題**：
- 講師多半自己在賣房，講座本身是銷售 funnel
- 高槓桿 = 風險放大：升息、租金違約、房屋折舊都會壓垮現金流
- 醫師本業已經很忙，當包租公的時間成本沒算進去

**判斷原則**：
- 房地產可以是配置一環，但**不該是主菜**
- 槓桿率 > 50% 的房產投資 = 高風險
- 不要被「醫師專屬」這個標籤打到防禦性

#### 雷 3：「複合式投資商品」

**情境**：銀行/券商推「結構債」「ELN」「KODA」「股權連動債」。

**問題**：
- 商品條款複雜，隱藏「下檔保護失效時 100% 虧損」條款
- 例：2008 雷曼連動債、2015 KODA 慘案
- 商品費率高，發行商穩賺不賠

**判斷原則**：
- 你看不懂的金融商品 = **不要買**
- 「保本」+「高收益」同時出現 = 條款一定有陷阱
- 純粹股票/債券 ETF 已經夠你用一輩子

### 🛡️ 醫師專屬防身術

#### 1. 五個「立即拒絕」關鍵字

| 關鍵字 | 一聽到就轉身 |
|---|---|
| 「**保證**」獲利 / 不會虧 | 🚨 違反金融鐵則 |
| 「**內線消息**」「未公開的好機會」 | 🚨 違法 + 多半假訊 |
| 「**今天最後一天**」「名額有限」 | 🚨 製造緊迫感 = 詐騙 SOP |
| 「**老師帶你**」「私訊私聊」 | 🚨 100% 詐騙 |
| 「**換 USDT/加密貨幣**入金」 | 🚨 追不到金流 |

#### 2. 三個「決策延遲」原則

- **24 小時規則**：任何投資決策**至少睡一晚再決定**
- **第二意見**：跟一個**無利益關係**的人討論（不是介紹人、不是賣商品的人）
- **網查 3 個關鍵字**：「公司名 + 詐騙」、「老師名 + PTT」、「平台名 + 評價」

#### 3. 一個「驗證合法」工具

| 想查 | 去哪查 |
|---|---|
| 投顧公司是否合法 | [中華民國證券投資信託暨顧問商業同業公會](https://www.sitca.org.tw/) |
| 銀行/券商是否合法 | [金管會證期局](https://www.sfb.gov.tw/) |
| 公司是否真實存在 | [經濟部商業司](https://findbiz.nat.gov.tw/) |
| 個股財報是否真實 | [公開資訊觀測站 mops.twse.com.tw](https://mops.twse.com.tw/) |

### 🆘 萬一被騙了怎麼辦？

#### 立即行動（黃金 24 小時）
1. **停損**：不要再轉錢進去（再多一塊都不要）
2. **保存證據**：對話紀錄、轉帳明細、平台網址全部截圖
3. **報警**：撥打 165 反詐騙專線
4. **通報銀行**：請銀行協助阻擋對方帳戶（黃金 24 小時內可凍結）

#### 後續
- 提告詐欺（刑事）+ 民事求償
- 尋找其他受害者一起集體訴訟
- **不要**自責太久——詐騙是專業犯罪，被騙不代表你笨

#### 預防再次受騙
- 一年內不要做任何新投資（讓自己冷靜）
- 跟家人/伴侶坦白（共同監督）
- 把錢全部放定存或大盤 ETF（**回到最簡單的策略**）

### 🎯 給自己的長期紀律

> 「我不需要找最賺的方法，我只需要**避開虧最慘的方法**。」

這條原則套用到醫師投資：

✅ **可以做的**：定期定額大盤 ETF / 全球分散 / 低成本被動投資
❌ **永遠不做**：聽明牌、加 LINE 群、買投資型保單、玩期權合約、買加密貨幣短線

20 年下來，光是避開上述陷阱，**就會贏過 95% 的醫師同行**。
