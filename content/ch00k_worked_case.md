## Ch0.10 範例實戰：完整分析一支股票

> 把前面所有名詞（EPS/ROE/ROA/本益比/殖利率/K線/均線/量/法人…）**實際用一遍**。
> 🔴 **重要**：下面「範例電子」「地雷光電」都是**我虛構的公司、數字是示範用**，不是真股票。
> 真的要分析時，把真實數字（去 Goodinfo 查）填進最後的空白模板即可。
> 框架來自 [Ch0.8 看盤大師決策框架](#00i)。

## 🟢 好例：範例電子（代號 9999）

### 數據表（示範數字）

| 項目 | 數字 | 類別 |
|---|---|---|
| 產業 | 半導體封測（長期需求向上） | 做什麼 |
| 股價 | 100 元 | — |
| EPS（近5年） | 4.0 → 4.8 → 5.5 → 6.2 → **7.0** | 品質 |
| ROE | 18% | 品質 |
| ROA | 11% | 品質 |
| 盈餘分配率 | 50%（配 3.5 元，來自盈餘） | 品質 |
| 本益比 | 100 ÷ 7 ≈ **14.3 倍**（同業平均約 18） | 估值 |
| 殖利率 | 3.5 ÷ 100 = **3.5%** | 估值 |
| 趨勢 | 收盤站上 20MA、60MA，季線上揚 | 技術 |
| 量價 | 上漲時量增 | 技術 |
| 法人 | 近期外資買超 | 籌碼 |
| 融資 | 沒暴增（散戶沒過熱） | 籌碼 |

### 用「四層漏斗」跑一遍

**第①層 大環境**：假設目前非全面恐慌/泡沫 → 過。

**第②層 體質（最重要）**
- EPS：**連 5 年正且逐年成長** → ✅ 真的在賺、而且越賺越多。
- ROE 18%（> 15%）→ ✅ 用股東的錢效率好。
- ROA 11%：ROE 18% vs ROA 11%，**差距不大** → ✅ 不是靠借錢硬撐（對照地雷例就懂）。
- 盈餘分配率 50%、配息來自盈餘 → ✅ 配得起、不是挖老本。
- **體質判定：健康 👍**

**第③層 估值**
- 本益比 14.3 倍 < 同業 18 倍，也不算歷史高檔 → 偏便宜。
- 殖利率 3.5% 是「**配很多**」來的（盈餘穩），不是股價跌出來的 → 非陷阱。
- **估值判定：合理偏便宜 👍**

**第④層 技術/籌碼（輔助）**
- 收盤 > 20MA > 60MA 且季線上揚 = 上升趨勢結構（[Ch0.7](#00h)）。
- 上漲量增 = 價量配合，較可信。
- 外資買超、融資沒過熱 = 籌碼偏穩。
- **技術判定：順勢、無明顯破壞 🔹**

### 檢查清單結果（[Ch0.8](#00i) 個股版）

```
✅ 能力圈：封測在做什麼，講得出來
✅ EPS 連續正且成長
✅ ROE>15% 且 ROA 不低
✅ 本益比比同業低
✅ 殖利率是配很多、非股價跌出來
✅ 趨勢沒崩壞
✅ 銀彈 5 年用不到
✅ 部位小、歸零也睡得著
→ 8/8 通過
```

> **看盤手結論**：範例電子在框架上是「**體質好 + 估值合理偏便宜 + 順勢**」的標的。
> ⚠️ 但這是**分析方法的示範**，不是叫你買它（它根本不存在）。就算真有這支，仍要**分批、小部位、只投閒錢**——因為「體質好 ≠ 一定會漲」。

## 🔴 地雷例：地雷光電（代號 6666）——學會看出壞的

### 數據表（示範數字）

| 項目 | 數字 | 紅旗 |
|---|---|---|
| EPS（近5年） | 3 → -2 → 1 → -4 → 0.5 | 🚩 忽正忽負、不穩 |
| ROE | **25%**（比好例還高！） | （別急著高興） |
| ROA | **3%** | 🚩 ROE 高 ROA 超低 = 借錢撐的（高槓桿） |
| 本益比 | 8 倍（好便宜？） | 🚩 是 EPS 不穩、市場不看好才被打低 |
| 殖利率 | **9%**（好高！） | 🚩 股價腰斬跌出來的 → 股息陷阱 |
| 趨勢 | 跌破季線、一波比一波低 | 🚩 下降趨勢 |
| 融資 | 近期暴增 | 🚩 散戶搶接刀、過熱 |

### 框架怎麼一眼看穿
- 新手直覺：「ROE 25% 最高、本益比 8 最便宜、殖利率 9% 最高 → 這支最好！」
- 看盤手：**全部是陷阱**——
  - ROE 25% 但 **ROA 只有 3%** → 高 ROE 是借錢撐的（[Ch0.6](#00g) 的灌水原理）。
  - 本益比 8 不是便宜，是 **EPS 忽正忽負**、市場不敢給高評價。
  - 殖利率 9% 是**股價暴跌**造成的假高（股息陷阱）。
  - 趨勢跌破季線 + 融資暴增 → 還在下墜、散戶在接刀。
- **檢查清單**：能力圈/EPS/ROE-ROA/趨勢… 多項打 ✗ → **直接跳過。**

> 🎯 **這就是框架的價值**：它讓你**不會被「表面數字漂亮」騙進地雷股**。
> 數字漂亮要問「**為什麼**漂亮」——這正是 [Ch0.6](#00g) 教的。

## ✍️ 換你動手：空白分析模板

拿一支**真股票**，去 Goodinfo 把數字填進來，自己跑一遍：

```
公司／代號：________  產業：________

【體質】
EPS 近5年：____（連續正且成長？ Y/N）
ROE：__%（>15%? ）  ROA：__%（ROE 高但 ROA 低 = 槓桿警訊？）
盈餘分配率：__%（配息來自盈餘？）

【估值】
本益比：__ 倍（vs 同業／自己歷史 → 貴/便宜？）
殖利率：__%（配很多？還是股價跌出來的？）

【技術（輔助）】
價 vs 20MA/60MA：____  趨勢：上升/下降/盤整？
量價配合？____

【籌碼（輔助）】
法人買超/賣超？____  融資有沒有暴增？____

【決策清單】通過 __/8 項 → 體質好不好？貴不貴？順不順勢？
→ 結論由你自己下；任何一項不安心 = 再等或跳過 + 分批小部位。
```
