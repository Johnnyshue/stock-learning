# 02 · XQ 全球贏家 XScript 選股／回測腳本生成器

> 複製自投資 YouTube 影片（`aXMBcJK9aeI`）的第 2 個功能：
> 用自然語言叫 AI 生成 XQ 的 XScript 選股腳本，並支援回測流程。

---

## 🔴🔴🔴 最重要的限制：語法為 best-effort，不保證正確

**本工具包的所有 XScript 程式碼都是 best-effort 範本，不是 XQ 官方保證語法。**

- 我（產出這些檔的 AI）**沒有 XQ 官方 XScript 語法手冊**。XScript 是
  XQ全球贏家（嘉實資訊）的專屬語言，確切函數名/欄位名/參數我無法 100% 驗證。
- **每一支腳本，你都必須在 XQ 編輯器按【編譯】驗證**，可能需依你實際的
  XScript 版本調整函數名。**在你按編譯通過之前，不保證能跑。**
- 凡是我不確定的函數（VIX 取值、月KD、本益比 PER、股價淨值比 PBR、現金殖利率、
  月營收 YoY、法人買賣超、選股輸出語法等），檔案裡都用 `// TODO: 確認 XQ 函數名`
  標出來了——這些就是「你最可能要在 XQ 裡修正」的地方。
- 這是刻意的誠實設計：**寧可標清楚不確定，也不給你一個看似完美卻編譯不過的腳本。**

### 編譯報錯怎麼辦
把 XQ 的紅字錯誤訊息（含行號）整段貼回給 Claude → 它幫你把該行 TODO 函數名改對 →
重編譯，來回幾次到綠燈。**這個來回是正常流程，不是失敗。**

## 怎麼用

1. **生成腳本**：跟 Claude 講你的選股條件（例：「VIX>40、月KD低檔、PBR低、殖利率>8%」），
   它依 `SKILL.md` 規則產出帶中文註解 + best-effort 警告的 XScript。
2. **貼進 XQ**：複製腳本 → XQ【選股中心/編輯器】貼上 → 按【編譯】。
3. **修錯**：報錯就把紅字貼回給 Claude 修 TODO 函數名，重編譯到通過。
4. **執行**：【加入選股】→ 跑出符合條件的股票清單。
5. **回測**：見 `回測workflow.md`。回測由 **XQ 軟體本身執行**，本工具包不代跑；
   你把 XQ 跑出的績效數字貼給 Claude，它可幫你整理多因子比較表。
